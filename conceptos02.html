Conceptos



Lo que debemos saber para trabajar con git y github


Terminos


Repositorio
Es un directorio donde se almacenan los archivos de un proyecto, y puede estar ubicado en el almacenamiento de Github o en el Repositorio local del computador, alli puedes almacenar todo lo referente a tu proyecto.

Clonar
Consiste en bajar una copia completa del repositorio en nuestro computador, y de esta manera poder tener toda la historia e informacion del repositorio a la mano.

Branch
Es una rama de tu repositorio donde puedes trabajar de manera aislada o puedes combinar tu rama con otras ramas.

Pull Request
Con el le informas a los demas que has enviado un cambio al Repositorio central.
Para hacerlo debes hacer lo siguiente:
-Vas al repositorio y buscas menu branch. -En el menu branch eliges la rama que tiene tu aporte. -Das click en New Pull Request. -Escribes el titulo y descripcion de tu pull request. -Finalmente pulsas en Create pull request.
